# She Can Foundation – NGO Landing Page

## Overview

This project is a simple and responsive NGO landing page created for the She Can Foundation Frontend Development Internship Task. The webpage aims to spread awareness about the foundation's mission of empowering women and promoting gender equality.

## Features

* NGO logo and header section
* About section describing the foundation's mission
* Interactive image hover effects
* Donation call-to-action banner
* Quick navigation cards
* Learn More and Certificate buttons
* Join Our Team section
* Contact information footer
* Social media links (Instagram and LinkedIn)
* Modern dark-themed design using HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Project Structure

* Header Section

  * NGO logo
  * Foundation name
  * Introduction and overview

* Donation Banner

  * Motivational message
  * Donate Now button

* Information Section

  * Our Story
  * Certificate
  * Donate

* Team Section

  * Volunteer recruitment information
  * Join Us button

* Footer

  * Contact details
  * Social media links

## Purpose

The purpose of this project is to create a clean and visually appealing webpage that highlights the mission and activities of the She Can Foundation while encouraging users to learn more, donate, and participate in community initiatives.
